# Plan 02-01: Utils Subpackage

## Goal
Create `uk_data/utils/` with promoted HTTP and timeseries helpers, shared `date_to_utc_datetime` / `encode_basic_auth` utilities, backward-compat re-exports, and unit tests.

- UTIL-01

## Tasks

- [ ] **Create `packages/uk-data/src/uk_data/utils/http.py`**
  - Copy all content from `_http.py` verbatim (same module docstring, same logic).
  - Keep the same public names: `get_json`, `get_text`, `get_bytes`, `retry`, `clear_cache`.
  - Add two new public helpers at the bottom:
    ```python
    def encode_basic_auth(user: str, password: str) -> str:
        """Return a Base64-encoded Basic auth token for *user*:*password*."""
        import base64
        return base64.b64encode(f"{user}:{password}".encode()).decode()
    ```
    (replaces `_encode_basic_auth` in `epc.py` and `api/client.py`)
  - Export `_USER_AGENT` as a module-level constant (it is already; keep it).

- [ ] **Create `packages/uk-data/src/uk_data/utils/timeseries.py`**

  - Copy `series_from_observations` factory (lines 89–127).
  - Copy `point_timeseries` factory (lines 130–162).
  - Add new public helper:
    ```python
    from datetime import UTC, date, datetime, time as _time

    def date_to_utc_datetime(d: date | datetime) -> datetime:
        """Coerce a bare *date* or a tz-naive *datetime* to a UTC-aware datetime.

        Mirrors the pattern used by ``_epc_event_timestamp`` (epc.py:248–250)
        and ``_transaction_event_timestamp`` (land_registry.py:232–235).
        """
        if isinstance(d, datetime):
            return d if d.tzinfo else d.replace(tzinfo=UTC)
        return datetime.combine(d, _time.min, tzinfo=UTC)
    ```
  - Keep `_parse_timestamp` module-private (leading underscore); all other symbols are public.

- [ ] **Add backward-compat re-imports to `_http.py`** (end of file, after `clear_cache`)
  ```python
  # Backward-compat: public surface is now uk_data.utils.http
  from uk_data.utils.http import (  # noqa: F401, E402
      clear_cache,
      encode_basic_auth,
      get_bytes,
      get_json,
      get_text,
      retry,
  )
  ```
  Note: `_http.py` retains its own implementation so it still works if imported directly before `utils/` exists. The re-import block should be wrapped in a `try/except ImportError: pass` to avoid circular-import issues during initial installation.

  **Correct approach** — do NOT duplicate the implementation. Instead, invert the dependency: `utils/http.py` holds all logic; `_http.py` becomes a thin shim:
  ```python
  # _http.py  — backward-compat shim; real implementation in utils/http.py
  from uk_data.utils.http import (  # noqa: F401
      _CACHE,
      _CACHE_LOCK,
      _CACHE_MAX_ENTRIES,
      _DEFAULT_TIMEOUT,
      _USER_AGENT,
      clear_cache,
      encode_basic_auth,
      get_bytes,
      get_json,
      get_text,
      retry,
  )
  ```
  Retain the module docstring from the original `_http.py`.
