| T-03-02 | I (Information Disclosure) | date-window filtering | mitigate | Enforce inclusive, deterministic filtering semantics in shared helper and lock with tests so callers cannot bypass range boundaries accidentally |
| T-03-03 | D (Denial of Service) | unbounded query paths | accept | Existing adapters already support `limit`; no new external query amplification introduced in this plan |
</threat_model>
